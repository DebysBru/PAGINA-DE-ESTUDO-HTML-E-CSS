HTML Básico

O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar páginas web. Não é uma linguagem de programação, mas sim de marcação, que define a estrutura e o conteúdo de uma página.

Estrutura Básica de um Documento HTML


<!DOCTYPE html>  
<html lang="pt-BR">  
<head>
    <meta charset="UTF-8">  
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título da Página</title>  
</head>
<body>
    <!-- Conteúdo visível -->
</body>
</html>

Como Criar um Arquivo HTML

  1. Abra um editor de texto (VS Code, Sublime, Notepad++)
  2. Crie um novo arquivo
  3. Salve com a extensão .html (ex: index.html)
  4. Abra no navegador para visualizar

CSS Básico

O que é CSS?

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para descrever a apresentação de um documento HTML. Enquanto o HTML estrutura o conteúdo, o CSS estiliza esse conteúdo.

Como Conectar CSS ao HTML

Existem três formas de adicionar CSS:

  1. Inline: Direto na tag HTML (não recomendado)
  2. Interno: Dentro da tag <style> no <head>
  3. Externo: Em um arquivo separado (recomendado)

Exemplo de CSS Externo

Crie um arquivo chamado estilos.css e adicione no HTML:


<head>
    <link rel="stylesheet" href="estilos.css">
</head>

Sintaxe Básica do CSS


seletor {
    propriedade: valor;
    outra-propriedade: valor;
}

Seletores Básicos

  • Seletor de elemento: p { color: blue; }
  • Seletor de classe: .destaque { background: yellow; }
  • Seletor de ID: #cabecalho { font-size: 24px; }

Box Model

Todos os elementos HTML são caixas com:

  • Conteúdo (content): O conteúdo real do elemento
  • Preenchimento (padding): Espaço entre o conteúdo e a borda
  • Borda (border): A borda ao redor do padding
  • Margem (margin): Espaço fora da borda
Conteúdo

Exemplo de Propriedades CSS


body {
    background-color: #f4f4f4; /* Cor de fundo */
    font-family: Arial, sans-serif; /* Fonte padrão */
    margin: 0; /* Remove margens padrão */
    padding: 20px; /* Espaçamento interno */
}

h1 {
    color: #2c3e50; /* Cor do texto */
    text-align: center; /* Alinhamento central */
    border-bottom: 2px solid #3498db; /* Borda inferior */
    padding-bottom: 10px; /* Espaçamento abaixo do texto */
}

.destaque {
    background-color: #ffffcc; /* Cor de fundo amarelo claro */
    padding: 10px; /* Espaçamento interno */
    border-left: 4px solid #f1c40f; /* Borda lateral esquerda */
}

Layouts com Flexbox

O que é Flexbox?

Flexbox é um modelo de layout CSS que permite organizar itens em um container de forma dinâmica, independentemente de seu tamanho. É especialmente útil para criar layouts responsivos.

Principais Propriedades do Flexbox

Para o container (parent):

  • display: flex; - Ativa o flexbox
  • flex-direction - Define a direção dos itens (row, column, row-reverse, column-reverse)
  • justify-content - Alinhamento horizontal (flex-start, center, flex-end, space-between, space-around)
  • align-items - Alinhamento vertical (flex-start, center, flex-end, stretch)
  • flex-wrap - Permite que os itens quebrem para a próxima linha

Para os itens (children):

  • order - Controla a ordem dos itens
  • flex-grow - Define como o item cresce em relação aos outros
  • flex-shrink - Define como o item encolhe em relação aos outros
  • align-self - Alinhamento individual para um item

Exemplo Prático de Flexbox

Item 1
Item 2
Item 3

HTML do Exemplo:


<div class="container">
    <div class="item">Item 1</div>
    <div class="item">Item 2</div>
    <div class="item">Item 3</div>
</div>

CSS do Exemplo:


.container {
    display: flex; /* Ativa o modelo flexbox */
    flex-direction: row; /* Disposição em linha */
    justify-content: space-between; /* Distribui espaço entre itens */
    align-items: center; /* Centraliza verticalmente */
    border: 1px solid #ccc; /* Borda cinza */
    padding: 10px; /* Espaçamento interno */
}

.item {
    background-color: #3498db; /* Cor de fundo azul */
    color: white; /* Texto branco */
    padding: 20px; /* Espaçamento interno */
    margin: 5px; /* Margem externa */
}

Exercício: Criando um Portfólio

Vamos criar uma estrutura básica para um portfólio usando HTML e CSS com Flexbox:

Estrutura HTML:


<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Meu Portfólio</title>
    <link rel="stylesheet" href="portfolio.css">
</head>
<body>
    <header class="cabecalho">
        <h1>Meu Portfólio</h1>
        <nav>
            <ul class="menu">
                <li><a href="#sobre">Sobre</a></li>
                <li><a href="#projetos">Projetos</a></li>
                <li><a href="#contato">Contato</a></li>
            </ul>
        </nav>
    </header>
    
    <main>
        <section id="sobre" class="secao">
            <h2>Sobre Mim</h2>
            <p>Texto sobre você...</p>
        </section>
        
        <section id="projetos" class="secao">
            <h2>Meus Projetos</h2>
            <div class="projetos-container">
                <div class="projeto">
                    <h3>Projeto 1</h3>
                    <p>Descrição do projeto...</p>
                </div>
                <div class="projeto">
                    <h3>Projeto 2</h3>
                    <p>Descrição do projeto...</p>
                </div>
            </div>
        </section>
    </main>
    
    <footer id="contato" class="rodape">
        <p>Entre em contato: email@exemplo.com</p>
    </footer>
</body>
</html>

CSS (portfolio.css):


/* Estilos gerais para o corpo da página */
body {
    font-family: Arial, sans-serif; /* Define a fonte padrão */
    line-height: 1.6; /* Espaçamento entre linhas */
    margin: 0; /* Remove margens padrão */
    padding: 0; /* Remove paddings padrão */
    color: #333; /* Cor do texto principal */
}

/* Estilização do cabeçalho */
.cabecalho {
    background-color: #2c3e50; /* Cor de fundo azul escuro */
    color: white; /* Cor do texto */
    padding: 1rem; /* Espaçamento interno */
    display: flex; /* Ativa flexbox */
    justify-content: space-between; /* Alinha itens nas extremidades */
    align-items: center; /* Centraliza verticalmente */
}

/* Estilização do menu de navegação */
.menu {
    display: flex; /* Layout flexível */
    list-style: none; /* Remove marcadores de lista */
    gap: 1rem; /* Espaço entre itens */
}

/* Estilização dos links do menu */
.menu a {
    color: white; /* Cor do texto */
    text-decoration: none; /* Remove sublinhado */
}

/* Estilização das seções */
.secao {
    padding: 2rem; /* Espaçamento interno */
    max-width: 1200px; /* Largura máxima */
    margin: 0 auto; /* Centraliza na página */
}

/* Container para os projetos */
.projetos-container {
    display: flex; /* Ativa flexbox */
    flex-wrap: wrap; /* Permite quebra de linha */
    gap: 1rem; /* Espaço entre itens */
    justify-content: center; /* Centraliza horizontalmente */
}

/* Estilização individual dos projetos */
.projeto {
    background-color: #f4f4f4; /* Cor de fundo cinza claro */
    padding: 1rem; /* Espaçamento interno */
    border-radius: 5px; /* Bordas arredondadas */
    flex: 1 1 300px; /* Cresce e encolhe com base de 300px */
    max-width: 400px; /* Largura máxima */
}

/* Estilização do rodapé */
.rodape {
    background-color: #2c3e50; /* Cor de fundo azul escuro */
    color: white; /* Cor do texto */
    text-align: center; /* Alinhamento central */
    padding: 1rem; /* Espaçamento interno */
    margin-top: 2rem; /* Margem superior */
}
